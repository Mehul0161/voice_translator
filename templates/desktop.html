Build Locally (Windows)

  1. Install Python 3.11 (recommended) and Git.
  2. Open PowerShell and run: 
    git clone https://github.com/Mehul0161/voice_translator
cd voice_translator/desktop
python -m venv venv
venv\Scripts\Activate.ps1
pip install -r requirements-desktop.txt
pip install pyinstaller
pyinstaller --noconsole --onefile main.py
  3. Find the EXE in dist/main.exe. Rename to LinguaSync.exe and share.

Troubleshooting

  • If audio doesn’t play, ensure speakers aren’t muted.
  • If microphone isn’t detected, check Windows privacy settings.
  • Install ffmpeg if needed for gTTS playback issues.
